#Descripción

Menú web responsive realizado únicamente con CSS 3. Primer prototipo para uso personal.

#Instrucciones
1) Si no lo tienes, instala la CLI de Gulp en su sistema

~~~
$ sudo npm install -g gulp-cli
~~~

2) Instala las dependencias

~~~
$ sudo npm install
~~~

3) Arranca gulp:

~~~
$ gulp
~~~
